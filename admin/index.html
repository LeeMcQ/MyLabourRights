How publishing works

  1. The backend stores nothing permanently (serverless is stateless).
  2. After publishing, use "Copy as data.js entry" and paste the object into the window.BLOG_POSTS array in your site source.
  3. Re-deploy the site — the article is now live for everyone.
  4. Social posting (LinkedIn / Facebook) happens at publish time, automatically, if those tokens are set in Netlify.
